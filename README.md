# 智能投稿选刊与期刊推荐skill

智能投稿选刊与期刊推荐技能。根据研究方向、职称评审要求、期刊收录偏好等条件，跨库检索知网、万方、维普及合作期刊平台，多维度评估并推荐最合适的投稿期刊。

## 功能

- 跨库检索知网、万方、维普及合作平台
- 5维度评分：方向匹配度 / 单位认可度 / 见刊时效 / 发表难度 / 性价比
- 投稿链接：经验证的期刊直接提供投稿入口，未验证或刊号不一致的期刊静默不展示
- 风险识别：假刊 / 套刊 / 增刊 / 灌水期刊等10+风险信号检测
- 输出HTML评估报告，含评分雷达图、对比表格、投稿按钮

## 投稿链接规则

- 仅在合作平台详情页完成**名称+CN号/ISSN双重比对**后，才展示投稿按钮
- 刊号不一致或不匹配的期刊，静默处理，不展示投稿链接、不展示任何提示
- 报告中仅在投稿按钮旁标注来源，其余位置不出现平台名称
