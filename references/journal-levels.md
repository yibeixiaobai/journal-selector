# 期刊级别分类标准

## 中国期刊分级体系

### 普通期刊（普刊）

| 类型 | 主管/主办单位 | 特点 |
|------|--------------|------|
| 国家级期刊 | 国家级部门/机构主管 | 如教育部、科技部等主管 |
| 省级期刊 | 省级部门/机构主管 | 如省教育厅、省社科院主管 |

**识别方式**：查看CN号中地区代码，非"11"（北京）开头的多为省级（但不绝对，需结合主办单位判断）。

### 核心期刊

| 名称 | 评选机构 | 周期 | 影响力 |
|------|---------|------|--------|
| 北大核心（中文核心） | 北京大学图书馆 | 每3年评选（2023版最新） | 国内最通用 |
| CSSCI（南大核心） | 南京大学中国社会科学评价中心 | 每2年评选 | 社科领域权威 |
| CSSCI扩展版 | 同上 | 同上 | 准核心级别 |
| 中国科技核心 | 中国科学技术信息研究所 | 每年评选 | 理工农医领域 |
| CSCD（中国科学引文数据库） | 中国科学院 | 每年评选 | 自然科学领域 |
| RCCSE（中国学术期刊评价） | 中国科学评价研究中心 | 每2年评选 | 综合评价体系 |
| 社科院AMI综合评价 | 中国社会科学院 | 每3年 | 人文社科领域 |

### 国际期刊

| 级别 | 说明 |
|------|------|
| SCI | 科学引文索引，自然科学领域顶级 |
| SSCI | 社会科学引文索引，社科领域顶级 |
| EI | 工程索引，工程技术领域重要 |
| CPCI | 科技会议录引文索引（原ISTP） |
| A&HCI | 艺术与人文引文索引 |
| Scopus | Elsevier旗下的综合数据库 |

## 期刊级别对职称评审的影响

### 职称评审常见要求

| 职称级别 | 通常要求的论文级别 | 备注 |
|---------|------------------|------|
| 初级→中级 | 普刊1-2篇 | 多数单位认可省级普刊 |
| 中级→副高 | 核心期刊1-2篇 或 普刊3篇 | 具体看单位评审文件 |
| 副高→正高 | 核心期刊2-3篇 | 部分要求CSSCI/SCI |

**重要提醒**：不同单位、不同省份的职称评审要求差异很大，选刊前务必确认用户所在单位的具体要求文件。

## 快速识别期刊级别

### 通过数据库名称标识识别

- **知网**：期刊详情页标注"北大核心""CSSCI""科技核心"等标签
- **万方**：期刊页面标注"核心期刊"分类
- **维普**：期刊页面标注收录级别

### 通过出版周期辅助判断

| 周期 | 通常对应 |
|------|---------|
| 周刊/旬刊 | 多为普刊，需警惕发文量过大 |
| 半月刊 | 普刊为主 |
| 月刊 | 普刊或部分核心 |
| 双月刊 | 多为核心或专业期刊 |
| 季刊 | 核心或权威期刊为主 |
| 半年刊 | 高级别专业期刊 |

### 通过影响因子辅助判断

| 影响因子范围 | 大致级别 |
|-------------|---------|
| <0.5 | 普刊或边缘核心 |
| 0.5-1.0 | 一般核心 |
| 1.0-3.0 | 中等核心 |
| 3.0-10.0 | 优秀核心 |
| >10.0 | 顶刊 |

## 期刊级别选择建议

### 按用途推荐

| 用途 | 推荐级别 | 说明 |
|------|---------|------|
| 职称评审（中级） | 省级/国家级普刊 | 三网收录，费用适中 |
| 职称评审（副高） | 北大核心/科技核心 | 视单位要求而定 |
| 职称评审（正高） | CSSCI/SCI/CSCD | 高级别核心 |
| 毕业论文 | 普刊或核心 | 视学校要求 |
| 项目结题 | 核心期刊 | 多数项目要求核心 |
| 学术交流 | 普刊即可 | 无级别硬性要求 |
| 奖励评定 | 核心期刊优先 | 高级别加分 |

### 按时间紧迫程度推荐

| 时间窗口 | 推荐级别 | 说明 |
|---------|---------|------|
| <1个月 | 审稿快的普刊 | 选择旬刊/月刊，录用快 |
| 1-3个月 | 省级/国家级普刊 | 选择月刊/双月刊 |
| 3-6个月 | 普刊或一般核心 | 核心审稿周期长 |
| >6个月 | 核心期刊 | 有充足时间打磨 |
