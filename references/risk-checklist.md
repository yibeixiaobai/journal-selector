# 非正规期刊识别清单与避坑指南

## 套刊/假刊识别

### 高风险信号

| 信号 | 说明 | 严重程度 |
|------|------|---------|
| 无CN号 | 正规期刊必有国内统一连续出版物号 | 🔴 致命 |
| CN号在新闻出版署查不到 | 套刊常用的假CN号 | 🔴 致命 |
| ISSN在ISSN国际中心查不到 | 假刊冒用或编造ISSN | 🔴 致命 |
| 期刊名称与数据库中不匹配 | 套刊常用相似名称 | 🔴 致命 |
| 承诺"百分百录用" | 正规期刊不可能保证 | 🟡 高风险 |
| 承诺"当天出录用通知" | 正规期刊审稿需要时间 | 🟡 高风险 |
| 价格远低于市场价 | 假刊常见诱饵（普刊正常1000-4000/版） | 🟡 高风险 |
| 无明确的主办/主管单位 | 正规期刊必须有 | 🟠 中风险 |
| 只在某个不知名网站收录 | 正规期刊应在知网/万方/维普中至少一个收录 | 🟠 中风险 |
| 联系方式只有微信/QQ | 正规期刊有编辑部电话和投稿邮箱 | 🟠 中风险 |

### 验证步骤

**第一步：查CN号**
- 访问国家新闻出版署：`https://www.nppa.gov.cn`
- 进入"期刊/期刊社查询"
- 输入CN号或期刊名验证

**第二步：查收录**
- 知网期刊导航：`https://navi.cnki.net/knavi`
- 万方期刊检索：`https://www.wanfangdata.com.cn`
- 维普期刊检索：`https://www.cqvip.com`

**第三步：交叉比对**
- 比对各数据库中期刊的主办单位、出版周期、ISSN是否一致
- 不一致则极有可能是套刊

**第四步：查目录**
- 在知网/万方查看最近2-3期目录
- 如果期刊声称录用但近期目录中无类似方向文章，需警惕

## 常见骗局模式

### 1. 套刊模式
> 伪造与正规期刊完全相同的封面、刊号，但实际为非法出版物。文章"发表"后不会被任何数据库收录。

**识别方法**：在知网搜索已发表的论文，如果搜不到则是套刊。

### 2. 增刊/专刊冒充正刊
> 部分机构将增刊、专刊当作正刊推荐，增刊虽合法但多数单位不认可。

**识别方法**：查看期刊封面标注，正刊标注"第X期"，增刊标注"增刊"或"S1"。

### 3. 电子刊冒充纸质刊
> 只有电子版（电子期刊号），没有纸质版（纸质CN号），部分单位不认可。

**识别方法**：区分CN号类型，纸质刊CN号格式为 `XX-XXXX/YYYY`，连续电子出版物为 `XX-XXXX/YY`。

### 4. 假冒网站投稿
> 伪造期刊投稿网站，收取"审稿费""版面费"后失联。

**识别方法**：
- 在知网期刊导航找到官方投稿链接
- 正规期刊投稿通常在采编平台（如 ScholarOne、玛格泰克）进行
- 不在正规投稿平台的要高度警惕

### 5. 中介虚报信息
> 部分中介虚构审稿周期、见刊时间，收取高额服务费后拖延或无法兑现。

**识别方法**：
- 通过官方渠道确认审稿周期
- 签订正规合同并保留付款凭证
- 不轻信口头承诺

## 版面费参考范围

| 期刊级别 | 每版费用（参考） | 备注 |
|---------|---------------|------|
| 省级普刊 | 800-2000元 | 地区差异大 |
| 国家级普刊 | 1500-3500元 | 部分高职称单位有补助 |
| 北大核心 | 3000-15000元 | 差异极大 |
| CSSCI | 5000-20000元 | 人文社科方向 |
| 科技核心 | 2000-8000元 | 理工农医方向 |
| SCI | 通常无版面费但有APC | 2000-20000美元不等 |

**注意**：以上为参考价格，实际费用以编辑部通知为准。价格显著低于参考范围时需警惕。

## 年发文量预警

| 年发文量 | 风险等级 | 说明 |
|---------|---------|------|
| <100篇 | 低风险 | 正常学术期刊 |
| 100-300篇 | 正常 | 大部分期刊在此范围 |
| 300-600篇 | 需关注 | 部分旬刊/半月刊正常范围 |
| 600-1000篇 | 高风险 | 可能存在灌水嫌疑 |
| >1000篇 | 极高风险 | 大概率为"论文工厂"期刊 |

## 投稿安全清单

在推荐期刊前，逐项检查：

- [ ] CN号真实有效（新闻出版署可查）
- [ ] ISSN真实有效（ISSN国际中心可查）
- [ ] 知网/万方/维普至少一网收录
- [ ] 主办单位为正规机构
- [ ] 年发文量在合理范围
- [ ] 投稿渠道为官方采编平台
- [ ] 版面费在合理范围
- [ ] 期刊不在预警名单中（如中科院预警期刊名单）
- [ ] 增刊/专刊已明确告知用户
- [ ] 收录数据库与用户单位认可要求匹配
